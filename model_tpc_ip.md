| Nazwa Warstwy | Opis |
| :-----------: | :--: |
| Warstwa Aplikacji | Działaja w niej urzyteczne dla urzytkownika aplikacje, np. przeglądarka internetowa |
| Warstwa Transportowa | Poprzez porty gwarantuje pewność połączenia i kieruje je do odpowiednich aplikacji |
| Warstwa Internetu | W tej warstwie ustalana jest droga do ostatecznego komputera w sieci |
| Warstwa dostępu do sieci | Warstwa ta zajmuje ssię przekazywaniem danych przez fizyczne połączenia między urządzeniami sieciowymi |
